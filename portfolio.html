WeatherJams

For our first project we as a team set out to create a weather dashboard called Weather Jams. The application is a simple user interface that lets users not only view the weather in their current location as well as other cities but it gives users the option to play music that corresponds with the current weather conditions. Using the SoundCloud API it generates playlists consisting of up to 400 songs. In addition to that we use the giphy api for the background which generates gifs based on the current weather. Altogether it makes for a unique user experience! You can see the fully deplyed application by clicking this link or by clicking the gif to the left.
